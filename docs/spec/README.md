## frozen/ について

`frozen/` 内のファイルは「凍結中・将来復活予定」の仕様。
現行UI（v4）には実装されていないが、将来的にコスト・優先度の問題が
解消されたタイミングで復活させる予定。

主な凍結項目：
- TravelMode（移動モード・地図ナビ）
  Google Maps APIのコストが課題のため保留中
